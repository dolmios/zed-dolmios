# Changelog

## 0.1.0

Initial release. Three variants: Light Soft, Dark Soft, Black.

Derived from Vitesse Soft with the following changes:
- Honey-amber accent replaces sage green on UI surfaces (selections, focused borders, link hover)
- Sage keyword/boolean/constructor tokens shifted to honey-amber (tag remains sage)
- Borders given explicit hierarchy (variant, selected, focused, transparent, disabled distinct)
- All text/icon/element null fields filled with explicit values
- Comments italic, alpha bumped from `dd` to `ee`
- Dark text bumped to full alpha (`#dbd7caee` -> `#dbd7ca`)
- Bright ANSI colors nudged ~10% lighter than base
- 6 warm-tone player colors populated
- Document highlights, focus borders, drop target, search match all populated
- Light Soft scrollbar track border fixed (`#fff` -> warm cream)
